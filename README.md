# FoxShareMyFile
A utility for convenient sending and receiving files without any restrictions!
You can easily share your files with anyone! Just rightâ€“click on the file, select the "Send" option - "FoxShareMyFile", share the link, and you're done!
There are no restrictions and third-party services (except in some cases).
